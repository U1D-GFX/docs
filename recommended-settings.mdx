## Display Settings

<ParamField path="Brightness" type="Display">
  Set brightness to **14 clicks from the left** on the in-game brightness slider. 
</ParamField>

## Graphics Settings

<ParamField path="Graphics Quality" type="Graphics">
  Set all graphics options to **High**.
</ParamField>

<Note>
  If you experience performance issues with all settings on High, try lowering **Shadows** and **Reflections** first, as these have the largest performance impact with the least visual tradeoff when using U1D GFX.
</Note>

## Quick Reference

| Setting           | Recommended Value   |
| ----------------- | ------------------- |
| Brightness        | 14 clicks from left |
| Graphics Preset   | High                |
| Shadows           | High                |
| Reflections       | High                |
| Ambient Occlusion | High                |
| Post FX           | High                |
