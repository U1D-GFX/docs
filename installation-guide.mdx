## Installation Steps

### 1. Navigate to your GTA V installation directory

Locate the folder where **GTA V** is installed. This folder contains files like `GTA5.exe`, `update`, and `x64`.

---

### 2. Choose your base type

Select the base that fits your setup:

- **No Plate Conflict Base** – Recommended for most servers to avoid plate texture conflicts.
- **Normal Base** – Standard version if your server does not require the conflict-safe version.

---

### 3. Install `.rpf` files to the FiveM mods folder

Copy the `.rpf` files from the selected base and place them inside the FiveM **mods** folder:

```text
C:\Users\USER\AppData\Local\FiveM\FiveM.app\mods
```

If the **mods** folder does not exist, create it manually.

---

### 4. Select and install an ENB preset

Choose an **ENB preset** that matches your system performance:

- **Performance Preset** – Best FPS for lower-end systems
- **Balanced Preset** – Good balance of visuals and performance
- **Ultra Preset** – Maximum graphics quality

Copy the preset files to your **GTA V main directory**.

---

### 5. Install ENB files

Install the ENB files in **two locations**:

#### GTA V Main Directory

Place the ENB files in the folder containing `GTA5.exe`.

Example files:

```text
d3d11.dll
enbseries.ini
enblocal.ini
```

#### FiveM Plugins Folder

Also copy the required files into the FiveM plugins folder:

```text
C:\Users\USER\AppData\Local\FiveM\FiveM.app\plugins
```

Create the folder if it does not already exist.

---

## Directory Reference

### Steam

Navigate to your Steam library, find **GTA V**, then:

**Right-click → Properties → Installed Files → Browse**

---

### Epic Games

Open the **Epic Games Launcher**, find **GTA V**, then click:

**View Installation Folder**

---

### Rockstar Games Launcher

Default installation path:

```text
C:\Program Files\Rockstar Games\Launcher
```

---

### FiveM Plugins Folder

```text
C:\Users\USER\AppData\Local\FiveM\FiveM.app\plugins
```

---

### FiveM Mods Folder

```text
C:\Users\USER\AppData\Local\FiveM\FiveM.app\mods
```

---

## ENB Configuration

After installation:

1. Launch **FiveM**
2. Join a server
3. If ReShade or ENB settings are available, press the configured key (commonly `HOME`) to open the menu
4. Select your preferred preset if prompted

---

## Notes

- Ensure **GTA V is closed** before copying files.
- Always **overwrite files if prompted**.
- Remove other graphics mods to avoid conflicts.

---
