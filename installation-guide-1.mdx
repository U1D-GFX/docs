### SFX Audio Installation

Step-by-step guide for installing custom audio and sound effects for GTA V.

---

### Installation Steps

1. Navigate to your GTA V main directory.
2. Locate the audio folders using the following path:

### x64 → audio → sfx

3. Drag and drop all audio files from the SFX download into the "sfx" folder.
4. If the audio does not work or causes issues, use the ArchiveFix tool for compatibility.

---

### Details

## Audio Path

### x64 → audio → sfx

ArchiveFix

Depending on your game version or installation, you may need to use ArchiveFix to ensure the custom audio files load correctly.

ArchiveFix can usually be found in the client download section where the audio pack or mod was downloaded.

---

### Important Notes

• Always back up original files before replacing any game files.

• Make sure GTA V is completely closed while installing the audio files.

• Restart the game after installation to ensure the sounds load properly.

• Some installations may require rebuilding the audio cache if the sounds do not appear in-game.

---

## Troubleshooting

### Audio not working:

- Double check the install path: x64 → audio → sfx
- Confirm files were placed in the correct folder
- Run ArchiveFix if provided with the download

Game crashes after install:

- Restore your backup files
- Reinstall the audio pack correctly
- Ensure the files are compatible with your GTA V build
