<Warning>
  U1D GFX Enhanced is **only compatible with GTA V Enhanced**. Do not install this on the legacy version of GTA V.
</Warning>

## Before You Begin

Make sure you have the following before installing:

- GTA V Enhanced installed and up to date
- Your GTA V Enhanced main game directory path ready
- The U1D GFX Enhanced pack downloaded from [u1dgfx.com](https://u1dgfx.com)

## Installation Steps

<Steps>
  <Step title="Copy the U1D GFX Files">
    Open the downloaded U1D GFX archive and locate the **U1D GFX** folder.

    Copy **all files** from inside the `U1D GFX` folder directly into your **GTA V Enhanced main game directory**.

    You should see a `mods` folder and a file named `rpf.cache.disable` appear in your game directory after copying.
  </Step>
  <Step title="Install the Requirements">
    Open the **Requirements** folder included in the download.

    Download and install the following into your **GTA V Enhanced main game directory**:

    - **ScriptHookV** — required for script support
    - **OpenRPF** — required for mod loading

    <Note>
      Both files must be placed in your GTA V Enhanced root directory alongside the game executable, not in a subfolder.
    </Note>
  </Step>
  <Step title="Launch the Game">
    Once all files are in place, you are ready to launch **GTA V Enhanced**.

    Start the game through your launcher as normal. U1D GFX will load automatically.
  </Step>
</Steps>

## Finding Your GTA V Enhanced Directory

Not sure where your game is installed? Common paths:

```
C:\Program Files\Rockstar Games\Grand Theft Auto V Enhanced\
```

You can also find it by right-clicking **GTA V Enhanced** in your launcher, selecting **Properties**, and clicking **Browse Local Files**.
